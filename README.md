# 功能描述

<img src=res/主界面.png>

一个通用的打包工具，可以应用于 Maya ma工程，Nuke 工程等非二进制工程文件打包

原理是通过正则匹配文本中可能的文件路径，根据文件名找到 UDIM，相关系列等文件，并保持目录结构保存到一个文件夹中。

文件释放时，用户可以自己设置路径对应关系

# 使用方法

## 打包

<img src=res/打包.png>

点击打包按钮，选个一个工程文件并设置输出目录点击打包按钮开始打包

## 释放

<img src=res/释放.png>

点击释放按钮，输入打包好的文件路径，点击解析，工具会分析源文件都存在于哪些目录，填写本地对应目录即可实现文件释放
